    Pawpd — API reference

    Quick jump: Client · Command/Cog · Message/Msg · Other

    pawpd.Client

    class Client(token, intents=None, prefix=None, builtin_help=True, logger=None)
    Main entry point. Connects to gateway, registers events/commands.

    Example: 

    from pawpd import Client
client = Client("TOKEN", prefix="!")
@client.event
async def on_ready(user):
    print("Ready:", user)
client.start()
